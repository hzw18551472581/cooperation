gitHub 相关操作命令行

1、拉取项目
    git clone '项目地址'
    (注意:会让你输入你的账号和密码，输入的过程中，确保正确输入，终端命令上是看不见你输入的账号密码的，输入完毕之后回车即可)
    (注意:克隆下来的项目会在本地创建一个与远程分支名对应的本地分支名，项目开发过程中创建自己的分支)
2、创建分支(星号常用)
     git branch                              --查看本地所有的分支
     git checkout 分支名                      --切换到本地存在的分支
    *git checkout -b 分支名                   --创建本地新分支并切换到该分支（克隆当前分支，创建一个新的本地分支，不需要手动切换分支）
    *git fetch origin 远程分支名：本地分支名    --创建一个与远程分支名对应的本地分支，并把远程分支代码拉取下来(需要手动切换分支)
    (注意：切换分支前需要保存当前分支的修改，否则会将修改的内容带到切换的分支上)
